# 🔢 MNIST Handwritten Digit Classification

## 📌 Project Overview
This project classifies handwritten digits (0-9) using 
three Machine Learning algorithms on the MNIST dataset 
from OpenML. Models are compared based on accuracy.

## 🏆 Model Results

| Model | Accuracy |
|-------|----------|
| 🌲 Random Forest | 92.67% ✅ Best |
| 🤖 KNN | 89.00% |
| 🌳 Decision Tree | 75.67% |

## 🔍 Key Findings
- Random Forest performed best with **92.67% accuracy**
- Decision Tree had lowest accuracy at **75.67%**
- KNN achieved solid **89.00% accuracy**
- Most misclassifications occur between similar digits
  like 4→9, 3→5, 8→1
